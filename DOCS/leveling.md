# Leveling TLDR

## XP Progression (Warrior Style)
- Max level: 35
- Early levels are fast, then ramp hard.
- XP thresholds (total XP required):
  - L1: 0
  - L2: 2,000
  - L3: 4,000
  - L4: 8,000
  - L5: 18,000
  - L6: 35,000
  - L7: 70,000
  - L8: 125,000
  - L9: 250,000
  - L10: 500,000
  - L11+: +250,000 per level
- L35 total XP: 6,750,000

## Perks (Permanent)
- L3: Token Magnet (+10% coins from games)
- L7: Extra Credit (+10% XP from games)
- L11: Arcade Allowance (minimum 20 coins per game)
- L15: Daily Double (+50% daily challenge coin rewards)
- L25: High Score Dividend (+1 bonus coin per 1,000 score)

## Special Milestone
- L20: CRT Overdrive
  - +20% coins, +20% XP
  - One-time bonus: 1,000 coins
